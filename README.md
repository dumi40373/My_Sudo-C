## Description

My-Sudo is a simplified re-implementation of the Unix `sudo` command written in C.
The project focuses on user privilege management and conditional command execution.

## Project Goals

- Understand user permission handling in Unix
- Execute commands conditionally
- Work with system calls
- Handle process execution and return codes
- Improve system-level programming skills

## Technologies Used

- Language: C
- Unix system calls
- Process execution
- Makefile

## Implemented Features

- Command execution with privilege validation
- Basic user verification
- Error handling
- Return status handling
